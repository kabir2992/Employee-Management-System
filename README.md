Employee Management System

A full-stack Role-Based Employee Management System built with the MERN stack, featuring authentication, OTP verification,
Admin Dashboard, CRUD operations, pagination, filtering, and role-based access control.

Features

Authentication
Signup with OTP Verification
Login with JWT Authentication
Role-based Access (Admin,Employer,User)
Protected Routes
Token-based authorization

Admin Dashboard
View All Users
View All Employers
View All Admin
Edit User Details
Delete User Details
View Detailed Info of Users
Highlight logged-in Admin
Pagination
Search & Filtering
Loading States
Toast Notification

Tech Stack

Frontend
React (Vite)
React Router DOM
Axios
Tailwind CSS
React Hot Toast
JWT Decode

Backend
Node.js
Express.js
